# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-01

### Added
- 🚀 Initial release of the Manoj Dahal futuristic portfolio
- 🎬 Cinematic loading screen with triple-ring AI loader
- 🖱 Custom cursor with dot, ring, glow halo, particle trail, click ripples
- 🌌 Multi-layer animated background (aurora, particles, AI grid, nebula, lightning)
- 🧭 Floating glass dock navigation with animated gradient border
- 📖 About section with timeline and animated stats
- 🧠 Skills section with 4 categories and 40+ glowing capsules
- 🪟 6 project cards with 3D tilt and glass borders
- ⚙️ 4-step process workflow
- 📬 Contact section with glass form (floating labels) and 4 info cards
- 🦶 Minimal footer with social icons
- 🪟 Project detail modal
- 📱 Fully responsive (desktop / tablet / mobile)

### Technical
- HTML5 semantic markup
- Modular CSS architecture (18 files: 1 entry + 1 variables + 1 base + 14 sections + 1 responsive)
- ES6 modules (16 JS files: 1 main + 1 data + 14 modules)
- Vanilla JavaScript — zero runtime dependencies
- Canvas API for particle systems
- IntersectionObserver for scroll reveals
- SCSS source files available for Sass users
- Deployment configs: Vercel, Netlify, GitHub Pages
- Dev tooling: build script, validation script, dev server
- SEO files: sitemap.xml, robots.txt, manifest.json
- GitHub workflow for CI/CD
